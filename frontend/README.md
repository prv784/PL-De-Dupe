# PL De-duplication System

A full-stack web application for de-duplicating Price List (PL) numbers from various file formats with user authentication, history tracking, and analytics.

## Features
- User authentication (Register/Login with JWT)
- File upload (CSV, Excel, JSON)
- Manual input of PL numbers
- De-duplication engine
- Download results in multiple formats
- History tracking
- Analytics dashboard
- Responsive design

## Technology Stack
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express.js
- **Database**: MongoDB
- **Authentication**: JWT + bcryptjs
- **File Processing**: XLSX, multer
